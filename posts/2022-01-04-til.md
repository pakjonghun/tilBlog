## 오늘 한 일

1. 백엔드 서버 반복해서 다시 작성(mvp 패턴과 인플릿트 방식으로 변경)
2. 프론트 사용자 인증 기능 작성

## 어려운점

1. 백엔드 를 온전히 다 작성하는데 시간이 너무 오래 걸린다.
2. 프론트 작성 할때 새로 업데이트 된 react router dom 사용이 익숙치 않아서 시간이 걸렸다.
3. 아직 react 작동 방식이 잘 이해가 되지 않는다.
4. 특히 상태에 따른 화면 변경 말고 다른 좋은 방법이 있을경우 (useNavigate Link 등)
5. 메모리 누수를 방지하기 위해 고려해야 할 것들이 좀 있는 것 같은데 아직 잘 감이 오지 않는다.
6. 쿠키사용이 아직 익숙치 않아서 이전에 한것 및 공식 홈페이지를 보고 작성했다.

## 해야 할 일

1. 잘 안되거나 익숙치 않는 부분 적어두고 틈틈히 볼 것(readmd->cookie multer json2csv use form 등)
2. 스테이트를 정리 안하고 바로 컴포넌트 를 이동시키거나(원래는 스테이트를 변경하고 그 변경을 감지해서 화면이 이동해야 정상임)
3. Link 컴포넌트에 onClick 와 to 프롭을 동시에 사용할 경우 고려해야 할점 판단해 볼 것
4. 슬슬 이력서와 포폴 준비를 해야 한다.(이력서는 조금씩 글로 작성 해 볼 것, 포폴은 계속 구상 할 것!! 기획한 것 다시 뜯어볼것)
